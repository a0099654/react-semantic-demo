## Folder Structure

After creation, your project should look like this:

```
my-app/
  README.md
  node_modules/
  package.json
  public/
    index.html
    favicon.ico
  src/
    components/
      App.js
      App.test.js
    styles/
      _shared.scss
      App.css
      App.scss
      index.css
    index.js
    index.test.js
    logo.svg
  .eslintrc
  .gitignore
  .stylelintrc
```
